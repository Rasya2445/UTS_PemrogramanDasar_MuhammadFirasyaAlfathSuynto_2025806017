# Soal 4 — Konversi CSV ke JSON

## Penjelasan
Soal 4 merupakan program Python yang digunakan untuk membaca file CSV hasil dari program C pada Soal 1, kemudian mengubah data tersebut menjadi format JSON.

Program Soal 4 berhasil:
- Membaca file CSV hasil program C
- Mengolah data mahasiswa
- Menghitung rata-rata nilai
- Mengubah format data menjadi JSON
- Menyimpan hasil konversi ke file data_mahasiswa.json

## Cara Run

bash
cd C\soal4_csv_json
python data_mahasiswa.py
